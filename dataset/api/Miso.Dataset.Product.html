Miso.Dataset. Product

new Product ( options )

A Miso.Product is a single computed value that can be obtained from a Miso.Dataset. When a dataset is syncable, it will be an object that one can subscribe to the changes of. Otherwise, it returns the actual computed value.

Parameters

  • options Object optional
    Properties
    • func function required

      the function that derives the computation

    • columns mixed required

      the columns from which the function derives the computation
